# Security Policy

## Reporting a Vulnerability

We take security and our users' trust seriously. If you believe you have found
a security issue in cnet, please responsibly disclose it by contacting
mybothelp@gmail.com, or by opening a private
[GitHub Security Advisory](https://github.com/G6938/cnet/security/advisories/new).

Please do not report security vulnerabilities through public GitHub issues,
discussions, or pull requests. Do not include live bot tokens, secrets, or
personal data in your report.
